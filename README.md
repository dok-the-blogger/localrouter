# Настройка обхода блокировок на роутере Asus (Entware)

Инструкция для восстановления конфигурации после сброса или замены железа.

## 1. Установка пакетов
Необходимо установить следующие пакеты через Entware:

```bash
opkg install autossh openssh-client curl
```

Если Xray не установлен вручную, его тоже нужно установить (например, из репозитория).

## 2. Копирование конфигов
Скопируйте файлы из репозитория в файловую систему роутера, сохраняя структуру каталогов:

- `/opt/etc/xray/config.json`
- `/opt/etc/init.d/S98do-tunnel`

Не забудьте заполнить заглушки (`YOUR_SELECTEL_IP`, `YOUR_SECRET_PASSWORD`, `YOUR_DO_DROPLET_IP`) реальными данными.

## 3. Права доступа и SSH ключ
**КРИТИЧНО:**

1.  Дайте права на выполнение скрипту туннеля:
    ```bash
    chmod +x /opt/etc/init.d/S98do-tunnel
    ```

2.  **SSH КЛЮЧ:**
    Скопируйте приватный ключ `do_ss` в `/opt/etc/ssh/`.
    **ВАЖНО:** Жестко ограничьте права на ключ, иначе SSH откажется работать:
    ```bash
    chmod 600 /opt/etc/ssh/do_ss
    ```
